+ **Project summary**: This project implements a book recommendation system using two distinct methods: Collaborative Filtering and Content-Based Filtering. The goal is to provide personalized book recommendations based on user preferences. A user-friendly Shiny app interface is developed, allowing users to input a book title and receive tailored book recommendations.
